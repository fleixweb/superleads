# Buyer Archetype Rules

Use buyer archetypes only when the user provides or the current evidence supports them. Do not make any archetype the default.

Generic archetype prompts:

- Importer or distributor: look for import, distribution, brands carried, territory, warehouse, reseller language.
- Wholesaler: look for bulk supply, B2B catalog, trade accounts, MOQ, dealer portals.
- Retailer or chain: look for stores, store locator, ecommerce catalog, buying categories.
- Brand owner: look for owned brands, private label, product families, trademark/brand pages.
- OEM or manufacturer: look for production capability, factory, contract manufacturing, certifications.
- End user or application site: look for usage context, facilities, projects, maintenance, procurement portals.
- Service contractor or integrator: look for installation, maintenance, aftermarket parts supply, project delivery, parts supply.

Each archetype is a hypothesis until supported by ClaimEvidence.
