# Product Profile Input

Collect only what helps the current task. Do not force a factory/OEM/industry questionnaire.

Useful fields:

- Product or service name.
- Target country or region.
- Target customer type or channel.
- Application context or downstream use.
- Keywords and excluded words.
- Seed companies, seed URLs, competitors, brands, or reference sites.
- Desired contact detail level.
- Target lead count and output mode.
- Must-verify claims.
- Existing table or source material paths.
